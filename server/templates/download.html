🌐 企业路由器 / 交换机 — SNMP 监控配置

如果您的设备无法安装 Agent 客户端（如企业级路由器、硬件防火墙、核心交换机），可通过 SNMP 协议接入平台监控。

📋 SNMP V2c 配置参数

SNMP 版本V2c
Communitypublic
目的主机82.156.229.67
⚠️ 确保 Community 为 public

📖 V2c 配置步骤

1. 登录路由器管理后台（通常为 192.168.1.1）

2. 找到「SNMP」或「网络管理」配置项

3. 启用 SNMP V2c，Community 填写 public

4. 配置 Trap/目的主机

5. 保存设置，重启 SNMP 服务

📌 配置完成后，在管理后台添加设备即可开始监控。

📋 SNMP V3 配置参数

SNMP 版本V3
安全级别authPriv
用户名自定义
认证协议MD5 / SHA
加密协议DES / AES
目的主机82.156.229.67
⚠️ 配置的用户名、认证密码、加密密码需与平台管理后台填写一致

📖 V3 配置步骤

1. 登录路由器管理后台（通常为 192.168.1.1）

2. 找到「SNMP」或「网络管理」配置项

3. 启用 SNMP V3，选择 authPriv 安全级别

4. 配置用户名、认证协议 (MD5/SHA) 及密码

5. 配置加密协议 (DES/AES) 及密码

6. 配置 Trap/目的主机

7. 保存设置，重启 SNMP 服务

📌 配置完成后，在管理后台添加设备，选择 V3 并填入对应信息。
