核心能力

不只是"通"或"不通"

传统监控只告诉你网络断了，LANWatch 告诉你断在哪、为什么断、以及怎么修。

🔍
断点诊断
离线后自动 traceroute，定位断在第几跳。客户打电话来说断网，你打开后台直接看到"断在 ISP 出口第 3 跳"。节省 80% 排查时间。
🗺️
静默拓扑发现
直接读本机 ARP 缓存表，3 秒内拿到局域网所有设备。不 ping 不发包，零噪音，不触发防火墙。
📡
多维度探测
同时探测网关 + 百度/阿里/腾讯/网易四路 DNS 延迟。真实反映网络质量，不只是"通"或"不通"。
🔔
微信即时报警
离线超过 3 分钟自动推送 Server酱 微信通知。防轰炸机制：同一故障 5 分钟内不重复提醒。
🔌
端口扫描与服务识别
扫描 43 个常见端口，自动识别 HTTP/SSH/SMB/DNS 等服务。30 线程并发，3-5 秒完成，每 5 分钟上报一次。
📱
手机端查看
微信扫码直接看，无需安装 App。每个设备独立二维码，管理员和企业主都能随时查看网络状态。
🔄
自动升级
后台静默检测 GitHub releases，有新版本弹窗询问是否升级。升级过程自动重启，零手动干预。
🌐
SNMP 监控
企业级路由器/交换机支持 SNMP V2c 协议接入，无需安装客户端，协议直连，后台统一管理。
