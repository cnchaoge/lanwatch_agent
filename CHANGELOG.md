## v1.0.0 正式版

**发布日期：** 2026-04-29

### 新增
- 智能诊断引擎：12 条诊断规则覆盖 Ping/HTTP/DNS/端口探测异常
- 引导式故障排查向导：5 种故障场景（网络不可达/高延迟/DNS 故障/HTTP 错误/服务宕机）
- 故障传播链分析：BFS/DFS 拓扑遍历，最长 3 跳传播路径
- 告警关联分析：300 秒时间窗口聚类，自动识别根因
- 前端 SPA 重构：暗色主题、移动端适配、响应式布局
- Web 引导向导页面：交互式故障排查（选择式/是非式/输入式步骤）
- 拓扑影响分析：展示节点故障对下游设备的影响范围
- 完整 API 参考文档、安装指南、FAQ、故障排查指南

### 优化
- 前端 UI 全面美化：CSS 变量主题系统、动效过渡、悬浮面板
- 移动端适配：汉堡菜单、触屏友好操作区域
- 代码结构模块化：诊断/向导/传播/拓扑各模块独立
- 服务健康检查脚本 `scripts/check_health.sh`
- 一键发布脚本 `scripts/release.sh`

### 技术
- 版本号 0.9.4 → 1.0.0
- FastAPI + SQLite3 + uvicorn
- 前端 Chart.js 图表渲染
- Python 3.8+ 兼容

## v0.7.0

**发布日期：** 2026-04-21

### 新增
- Windows 客户端正式版发布
- 支持多网段、多目标配置
- 系统托盘运行，开机自启
- 托盘菜单（打开监控面板/检查更新/卸载/关于）
- 注册窗口配置（企业名称/联系电话/地址/网段/目标）
- 安装注册完成显示二维码（手机端扫码查看）
- 开机自启注册表设置
- 自动卸载功能（通知服务端后清理配置）
- ARP 局域网拓扑扫描（MAC 厂商识别）
- 端口扫描与服务识别（HTTP/SSH/SMB/DNS 等）
- 断点诊断（traceroute 到目标）
- 离线 3 分钟自动 Server酱 微信报警
- 多维度探测（网关 + 百度/阿里/腾讯/网易 DNS）
- 探测数据每 60 秒上报，拓扑每 5 分钟上报
- 云端存储，Web 后台查看
- 手机端扫码直接查看（无需 App）
- 微信扫码登录（功能开发中）
