🚀 安装步骤

1
上传文件到路由器
使用 scp 将整个 router 目录上传到路由器的 /tmp/lanwatch
# 在本地 Mac/Linux 执行 scp -r agent/router root@192.168.8.1:/tmp/lanwatch # 然后 SSH 登录路由器 ssh root@192.168.8.1
2
执行安装脚本
cd /tmp/lanwatch sh install.sh
3
配置企业名称并启动
uci set lanwatch.main.company_name='我的企业' uci commit lanwatch /etc/init.d/lanwatch start
⚠️ 注意：安装前请确保路由器已安装 curl。如未安装，执行 opkg update && opkg install curl
